MiracleList ist eine Beispielanwendung, angelehnt an die Webanwendung und App "Wunderlist", die die Firma Microsoft im Jahr 2015 für über 100 Millionen Dollar kaufte und dann im Jahr 2020 zugunsten von "Microsoft ToDo" beerdigte.

Dieser Server stellt das Backend auf Basis von .NET/C#/ASP.NET Core bereit. Es ist kostenfrei für Test- und Ausbildungszwecke für jedermann nutzbar. Eine Registrierung für eine Client-ID ist aber erforderlich, damit jeder Client einen eigenen Mandaten erhält.
